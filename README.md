# Inoko 
🐗 is highly opinionated static site generator framework.


## 🐗 installation
Install via your favourite package manager. Eg. `npm i inoko`.


## Templating
🐗 uses pugjs, as templating engine

## Style
🐗 uses Stylus as CSS preprocessor
